# rainbow-penguin

A npm library to send motivational messages to developers

# Installation

```sh
$ npm i rainbow-penguin
```

# Usage

```js
require('rainbow-penguin')();
```

# Enabling/Disabling

```js
const rainbowPenguin = require('rainbow-penguin');

if (process.env.NODE_ENV == 'development') rainbowPenguin();
```
